# web-code
我构建了一个基于多 Agent 协作的小米智能家居意图理解与自动化系统。核心痛点：现有智能家居控制依赖用户逐设备精确操作，缺乏对模糊意图的深层理解能力，导致"我要睡了"这类自然指令无法被正确拆解执行。  系统架构包含三层 Agent 协作：意图解析 Agent 负责将模糊自然语言指令通过长链推理拆解为结构化子任务链（如"睡觉"→灯光关闭、空调调至26度睡眠模式、窗帘关闭、安防启动）；设备调度 Agent 根据子任务匹配米家生态设备并生成执行序列，处理设备间依赖关系与冲突消解（如空调和加湿器的联动策略）；反馈验证 Agent 监控执行结果，若设备离线或执行失败则触发降级策略并通知用户。
